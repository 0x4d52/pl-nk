Detailed Description

A Deleter which doesn't actually delete anything.

This Deleter just leaks memory by ignoring requests to delete anything. This is probably pretty useless but could be used to test the effect of a real memory leak over a long period. Initially it was used to isolate a problem where objects being deleted were causing a crash. This class was used so that the objects weren't actually deleted so the problem (which is now fixed) could be put to one side.

Obviously use this with caution!
