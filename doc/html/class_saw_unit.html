Detailed Description

template<class SampleType>
class SawUnit< SampleType >

A non-bandlimted sawtooth oscillator.

Although the template class name is SawUnit the abbreviated name Saw may be used when employing the deafult sample type (which defaults to float).

Examples: 

 Unit test()
 {
    return Saw::ar (440, 0.1); // 440Hz, amplitude 0.1
 } 

 Unit test()
 {
    // Modulated frequency rising 600-900Hz, amplitude 0.1
    return Saw::ar (Saw::ar (1, 300, 600), 0.1); 
 }
