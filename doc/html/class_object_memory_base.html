Detailed Description

Manage a custom memory allocation system.

An ObjectMemoryBase subclass manages a Memory instance. Care should be take for this to be created before any other objects EXCEPT for Memory instances and AudioHostBase subclasses. In fact the AudioHostBase provides a means for using an ObjectMemoryBase subclass and controls its creation and running.
